SHORT DESCRIPTION
    This module contains DSC resources for configuration of the WSMan Credential Security Support Provider (CredSSP).

LONG DESCRIPTION
    This module contains DSC resources for configuration of the WSMan Credential Security Support Provider (CredSSP).

EXAMPLES
